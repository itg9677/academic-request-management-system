# نظام الخدمات الطلابية (Student Services System)

نظام إلكتروني يتيح للطالب تقديم طلباته الأكاديمية والإدارية إلكترونياً، ويتيح لموظفي القسم والشؤون الطلابية معالجتها ومتابعتها بسهولة.

## 🚀 مبني باستخدام
- Firebase Hosting

## ✨ المميزات
- تقديم طلب حذف/إضافة مواد دراسية
- رفع الأعذار الطبية للتغيب عن الاختبارات
- طلب زيارة مقرات أخرى
- رفع الشكاوى والاقتراحات
- لوحة تحكم لموظفي القسم والشؤون الطلابية لمعالجة الطلبات ومتابعة حالتها

## 🌐 الديمو
المشروع منشور على:
👉 [students-req.web.app](https://students-req.web.app)

## 📦 النشر (Deployment)
المشروع منشور باستخدام Firebase Hosting:

```bash
npm run build
firebase deploy
```

## 👥 المستخدمون
- **الطالب:** يقدم الطلبات ويتابع حالتها
- **موظف القسم / الشؤون الطلابية:** يستقبل الطلبات، يراجعها، ويوافق أو يرفض
- **مدير النظام:** يتابع جميع الطلبات، ويشرف على أداء الموظفين

## 🛠️ التقنيات المستخدمة
- Hosting: Firebase Hosting
